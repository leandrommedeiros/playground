# 1. Introduction

The **KatuOS API Quick Reference** is a compact reference manual for the public
KatuOS programming interface.

It is intended for day-to-day application development and provides a concise
description of every public function exposed by the kernel.

Unlike the *Scheduling, Delays & Synchronization Manual*, which explains the
kernel architecture, execution model and design rationale, this document focuses
exclusively on the public API.

Each API entry contains:

- a brief function description;
- the complete function prototype;
- parameter descriptions;
- return value information;
- important usage remarks;
- a minimal usage example;
- references to related APIs.

Only the public application interface is documented here.

Processor-specific implementation details and internal kernel services are
described in their respective technical manuals.
