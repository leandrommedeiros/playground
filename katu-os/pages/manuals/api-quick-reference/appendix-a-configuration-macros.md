# Appendix A — Configuration Macros

This appendix summarizes the compile-time configuration macros available in
KatuOS.

| Macro | Description |
|-------|-------------|
| `MAX_TASKS` | Maximum number of application tasks. |
| `MAX_TIMERS` | Maximum number of software timers. |
| `ENABLE_TIMERS` | Enables the Software Timer service. |
| `ENABLE_SEMAPHORES` | Enables semaphore support. |
| `ENABLE_QUEUES` | Enables queue support. |
| `ENABLE_CPU_LOAD_MONITOR` | Enables CPU Load monitoring. |
| `ENABLE_STACK_PATTERN` | Enables stack pattern initialization and telemetry. |
| `ENABLE_MAINTENANCE_PROCESS` | Enables the Kernel Maintenance Process. |

>  
> See the **Configuration Guide** for a complete description of each option.
