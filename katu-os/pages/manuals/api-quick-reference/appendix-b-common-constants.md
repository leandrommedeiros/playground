# Appendix B — Common Constants

The following constants are commonly used throughout the public API.

| Constant | Description |
|----------|-------------|
| `OS_WAIT_FOREVER` | Wait indefinitely. |
| `DELAY_LOCK` | Reserved for internal kernel use. |
| `true` | Boolean true. |
| `false` | Boolean false. |
| `NULL` | Null pointer. |

> Additional constants may be defined by individual kernel modules.
